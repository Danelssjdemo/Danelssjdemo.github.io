Node.js安装

Node.js是一个基于Chrome Javascript运行时建立的一个平台，用来方便的搭建快速的，易于扩展的网络应用。

Node.js借助事件驱动，非阻塞I/O模型变得轻量和高效，非常适合run across distributed devices的data-intensivede 的实时应用。

在 Linux 中安装 Nodejs

安装完成之后，可以通过下面的命令来验证一下Node.js是否安装成功。

$ node -v v0.10.33
